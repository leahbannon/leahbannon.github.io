About APIs

  1. This is such a big umbrella term that basically means anything that lets computers talk to each other
  2. You need to know about them because they are fundamental to much of how the web works
  3. Some types: basic website, oauth logins, data
  4. I think it's a good foundation for starting to get into tech because it opens up the tech part of your problem solving brain
  5. Important: always build on top of an API first, i.e. “dogfooding"
